# numberSumDiviability
Check if the sum of each number is diviable by divisor

Description:
returns array of numbers between start and stop, if sum of each number is diviable by divisor
diviable_by(start, stop, divisor)

Example:

diviable_by(10, 20, 5)
returns: ['14', '19']
14 = (1 + 4) / 5 
19 = (1 + 9) / 5


Language: Python
